1 流程

1.1 预处理

1.1.1 netconfig

$H7/v0.1/netconfigdev?user=&ver=1&channel=

1.1.2 检查更新

$H2/v0.1/static/ppt/finalupdate/update/version.dat?id=0144BA9D 返回格式: 文件下载(内容只有一个版本代码)

1.1.3 encode

$8/v0.1/c/math/831989148/info 返回格式: 点击查看 endode

1.1.4 code

$8/v0.1/c/math/831989148/code 返回格式: 点击查看 checkcode

1.1.5 checkcode

$8/v0.1/c/math/831989148/check?checkcode=BI8TuYp9 返回格式: 点击查看 checkcode

1.2 选教材

1.2.1 年级

$H1/v0.6/categories/relations?patternPath=K12 返回格式: 点击查看 年级列表

1.2.2 学科

$H1/v0.6/categories/relations?patternPath=K12/$ON020300(年级代码)/ 返回格式: 点击查看 八年级学科列表(json)

1.2.3 教材版本/子版本

$H1/v0.6/categories/relations?patternPath=K12/$(年级代码)/$(学科代码) 返回格式: 点击查看 八年级英语教材版本与子版本

1.2.4 获取书籍信息

$H1/v0.6/teachingmaterials/actions/query?&include=TI,EDU,CG&category=K12/$ON030000/$ON030200/$SB0400/$E001000/$E001001&relation&coverage=Org/nd/&prop&words&limit=(0,20) 返回格式: 点击查看 书籍信息

1.3 选章节

1.3.1 章节

$H1/v0.6/teachingmaterials/af0b4778-3531-4614-a65c-07c90d7839a5/chapters/none/subitems 返回格式: 点击查看 八年级英语人教上册章节列表

1.4 获取资源

1.4.1 课件

$H1/v0.6/coursewares/actions/query?words&limit=(0,1)&include=TI,LC&category&relation=chapters/a488f154-d2ea-4ec1-b520-c74dc1a04aab/ASSOCIATE&coverage=Org/nd/&category=$F010003&category=$F060005&category=$F010004 返回格式: 点击查看 课件资源

1.4.2 ebook

$H1/v0.6/ebooks/actions/query?words&limit=(0,1)&include=TI,CG,LC&relation=teachingmaterials/af0b4778-3531-4614-a65c-07c90d7839a5/ASSOCIATE&coverage=Org/nd/ 返回格式: 点击查看 ebook资源

1.4.3 多媒体资源

$H1/v0.6/assets/actions/query?words&limit=(0,20)&include=TI&category=$RA0104&relation=chapters/a488f154-d2ea-4ec1-b520-c74dc1a04aab/ASSOCIATE&coverage=Org/nd/ 返回格式: 点击查看 多媒体资源 (其中limit控制返回数量, category控制多媒体类型:见附录, 可添加 categoryexclude参数 e.g: categoryexclude=$F020004, 用于)

1.4.4 3D资源

$H1/v0.6/assets/actions/query?words&limit=(0,1)&include=TI,LC&category=$RT0209%20and%20$F070004&category=$RA0106%20and%20$F070004&category=$RA0106%20and%20$F080001&relation=chapters/a488f154-d2ea-4ec1-b520-c74dc1a04aab/ASSOCIATE&coverage=Org/nd/&orderby=viplevel%20desc 返回格式: 点击查看 3D资源

1.4.5 基础习题

$H1/v0.6/questions/actions/query?words&limit=(0,1)&include=TI,CG,LC&category=$RE0200&categoryexclude=$RE0211,$RE0206,$RE0217&relation=chapters/a488f154-d2ea-4ec1-b520-c74dc1a04aab/ASSOCIATE&prop=status%20ne%20CREATING&coverage=Org/nd/ 返回格式: 点击查看 基础习题

1.4.6 趣味题型

$H1/v0.6/coursewareobjects/actions/query?words&limit=(0,1)&include=TI,CG,LC&relation=chapters/a488f154-d2ea-4ec1-b520-c74dc1a04aab/ASSOCIATE&category=$RE04*&prop=status%20ne%20CREATING&coverage=Org/nd/ 返回格式: 点击查看 趣味题型

1.4.7 PPT底板

$H1/v0.6/assets/actions/query?words&limit=(0,1)&include=TI,LC&category=$RA0501%20and%20$F050003&category&coverage=Org/nd/ 返回格式: 点击查看 PPT底板

1.4.8 VR测试资源(未投入使用)

$H1/v0.6/assets/actions/query?words&limit=(0,1)&include=TI,LC&category=$RT0400%20and%20$F070004&category=$RT0402&category=$RT0403&relation&coverage=Org/nd/&orderby=viplevel%20desc 返回格式: 点击查看 VR测试资源

1.4.9 未知用途资源

$H1/v0.6/coursewares/actions/query?words&limit=(0,1)&include=TI,LC&coverage=Org/nd/&relation&category=$RT0101 返回格式: 点击查看: 未知用途
